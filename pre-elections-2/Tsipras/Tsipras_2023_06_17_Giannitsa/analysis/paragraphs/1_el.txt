Σας ευχαριστώ θερμά γι'αυτή τη ζεστή, την πολύ ζεστή υποδοχή που μου επιφυλάξατε σήμερα εδώ πολίτες των Γιαννιτσών και της Πέλλας. Σας ευχαριστώ θερμά. Είναι για μένα μια πολύ συγκινητική στιγμή να σας βλέπω όλους εσάς εδώ μια βδομάδα πριν από μια κρίσιμη εκλογική αναμέτρηση, να δίνετε το μήνυμα «επιτρέπεται να πέσεις, επιβάλλεται να σηκωθείς και να παλέψεις». Θα παλέψουμε μέχρι το τελευταίο δευτερόλεπτο, μέχρι να κλείσει η κάλπη για να ανατρέψουμε αυτόν τον συσχετισμό. Θα παλέψουμε όχι για λογαριασμό του δικού μας συμφέροντος του κομματικού, αλλά για λογαριασμό της κοινωνίας που έχει ανάγκη έναν ΣΥΡΙΖΑ ισχυρό, που έχει ανάγκη να είναι δυνατή, να είναι δυνατός ο πυλώνας της προοδευτικής παράταξης του τόπου και στις εκλογές και την επόμενη μέρα. Και θέλω σήμερα εδώ απέναντί σας να δεσμευτώ για ένα πράγμα: Θα πάμε μαζί να δώσουμε αυτή τη μάχη μέχρι την κάλπη, αποφασιστικά δυνατά, με αισιοδοξία και θα είμαστε μαζί και στις 25 αλλά και στις 26 του μήνα. Θα συνεχίσουμε μαζί να αγωνιζόμαστε για τις ανάγκες της κοινωνίας, να αγωνιζόμαστε για τα συμφέροντα και τα δικαιώματα των εργαζομένων, των αγροτών, των παραγωγών, των μικρομεσαίων επαγγελματιών, για τα συμφέροντα των μη προνομιούχων Ελλήνων. Για αυτό γεννήθηκε ο ΣΥΡΙΖΑ, για αυτό είναι εδώ ο ΣΥΡΙΖΑ-Προοδευτική Συμμαχία, γι' αυτό θα είναι και αύριο εδώ. Θέλω να ξέρετε ένα πράγμα. Ο ΣΥΡΙΖΑ-Προοδευτική Συμμαχία δεν είναι ούτε θα γίνει κόμμα διαμαρτυρίας. Δεν είναι ούτε θα γίνει συμπλήρωμα της κυβερνητικής εξουσίας των πολιτικών μας αντιπάλων, της Δεξιάς. Ο ΣΥΡΙΖΑ είναι και θα παραμείνει κόμμα εξουσίας. Θα διεκδικήσουμε κάθε ελπίδα, κάθε δυνατότητα σε αυτή την κάλπη της 25ης του Ιούνη αλλά θα είμαστε εδώ σε κάθε περίπτωση για να ξανακάνουμε πολύ σύντομα μεγάλη, δυνατή την προοδευτική παράταξη με πυλώνα τον ΣΥΡΙΖΑ-Προοδευτική Συμμαχία για να φέρουμε σε αυτό τον τόπο προοπτική, ελπίδα και την αλλαγή την κυβερνητική που είναι αναγκαιότητα, είναι αναγκαιότητα κοινωνική. Θέλω πολίτες της Πέλλας, των Γιαννιτσών, να σας πω και της ευρύτερης περιοχής εδώ της Πέλλας και της Έδεσσας -ήμουνα πριν από λίγες μέρες εκεί- θέλω να σας πω ότι έχουμε αποφασίσει να θέσουμε σε αυτήν την προεκλογική περίοδο επί τάπητος τα μεγάλα προβλήματα που αντιμετωπίζει η ελληνική κοινωνία.
