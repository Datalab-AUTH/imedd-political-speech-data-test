Να μιλήσουμε για τα μεγάλα προβλήματα τα οποία αντιμετωπίζει σήμερα, αλλά κυρίως για τις λύσεις. Τις λύσεις που προτείνουμε μεσοπρόθεσμα και μακροπρόθεσμα για να μπορέσουμε να στηρίξουμε τη μικρομεσαία επιχειρηματικότητα, τους ανθρώπους της εργασίας, να αναδείξουμε τα ανταγωνιστικά πλεονεκτήματα των τοπικών οικονομιών, να προχωρήσουμε στην παραγωγική ανασυγκρότηση προς ένα νέο μοντέλο παραγωγικό, που θα αξιοποιεί αυτές τις δυνατότητες κάθε τόπου ξεχωριστά και θα έχει μια προοπτική, έναν ορίζοντα, το όραμά μας: μια δίκαιη κοινωνία και ευημερία για όλες και για όλους.
