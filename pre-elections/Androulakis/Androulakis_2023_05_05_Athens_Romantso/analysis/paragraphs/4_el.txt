Στόχος μας είναι να διασφαλίσουμε μια ισχυρή βιώσιμη ανάπτυξη που θα δίνει δυνατότητα σε περιφερειακό επίπεδο να σταθεί η οικονομία όρθια στα πόδια της με ανταγωνιστικότητα. Με εσωστρέφεια με αξιοποίηση των σύγχρονων ψηφιακών δυνατοτήτων σε όλους τους τομείς παραγωγής και πάντα με σεβασμό στο περιβάλλον. Τα οφέλη αυτής της πολιτικής δεν μπορούν να είναι μόνο για τους λίγους και τους ισχυρούς πρέπει να διαχέονται στο σύνολο της ελληνικής κοινωνίας. Για αυτό μιλάμε για ένα προοδευτικό φορολογικό σύστημα που εξασφαλίζει τους απαραίτητους πόρους για τη μείωση των ανισοτήτων και την καταπολέμηση της φτώχειας, ανάπτυξη που συνδέεται με τις πραγματικές ανάγκες των ανθρώπων που εγγυάται μια καλύτερη ζωή για τον εργαζόμενο, ώστε να ζει σε ένα σπίτι που μπορεί να πληρώσει πρόσβαση σε στέγη. Με αξιοπρέπεια και χαμηλό κόστος να πάρει μετά από αρκετά χρόνια ζωής μια αξιοπρεπή σύνταξη και να νιώθει ασφάλεια σε ένα σύστημα υγείας που τον στηρίζει στα δύσκολα και όχι όπως σήμερα που ζούμε. Μια ολική παρακμή. Με πολιτικές που στηρίζουν την ελληνική οικογένεια για να μπορεί να τα βγάλει πέρα, με κοινωνικές δομές και ποιοτικό δημόσιο σύστημα εκπαίδευσης για όλα τα παιδιά του ελληνικού λαού. Στόχος μας είναι να προσφέρουμε στους νέους ανθρώπους δίκαιες και ίσες ευκαιρίες για να ζήσουν, να εξελιχθούν στον τόπο τους, να μορφωθούν, να δουλέψουν με αξιοπρεπείς μισθούς, να κάνουν οικογένεια.
