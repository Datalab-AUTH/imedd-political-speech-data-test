Παίρνω την ευκαιρία επειδή σήμερα είναι μια συγκέντρωση της νεολαίας να μην κάνω, όπως βλέπετε, μια παραδοσιακή προεκλογική. Συζήτηση συζήτηση ομιλία. Γιατί μου δώσατε αυτή την ευκαιρία και σας ευχαριστώ πάρα πολύ για αυτό. Γιατί οι εκλογές για μας είναι μια ευκαιρία, δεν είναι αυτοσκοπός. Το μέρα 25 συμμαχία για τη ρήξη χρησιμοποιούμε τις εκλογές. Όχι ως στόχο. Αλλά ως μέσο. Για να χτίσουμε αυτό που έχει ανάγκη η νεολαία μας τη νέα μεγάλη αγωνιστική αριστερά που θα βάλει τις βάσεις. Για να μπορείτε να σκεφτείτε πώς θα κατακτήσετε την εξουσία. Να μπορείτε να σκεφτείτε πώς θα πάρετε τα μέσα παραγωγής στα χέρια σας, έτσι ώστε να απελευθερωθείτε. Όχι από την τεχνολογία, αλλά από εκείνους. Που έχουν στα χέρια τους. Την ιδιοκτησία της τεχνολογίας που πάντοτε στρεφόταν εναντίον των μη εχόντων. Αυτά τα ιδιοκτησιακά δικαιώματα. Αυτό σημαίνει αγωνιστική αριστερά. Όλη αυτή η διαδικασία της απαξίωσης, γιατί αυτό είναι απαξίωση. Μιας και είπα τη λέξη απαξίωση.
