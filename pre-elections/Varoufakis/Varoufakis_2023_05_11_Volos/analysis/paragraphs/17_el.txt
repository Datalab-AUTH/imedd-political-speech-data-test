Εμείς τι λέμε;Επιτρέψτε μου να. Απαντήσω. Στο ρητορικό αυτό ερώτημα, με μια αναφορά σε κάτι που είπε ο Αλέξης Τσίπρας στο ντιμπέιτ, κάποια στιγμή; Σε κάποια στιγμή πέταξε μια ατάκα ότι αυτό που χρειάζεται για την αριστερά, το ακούσατε, είναι ενότητα. Συμφωνούμε. Αλλά ενότητα για να κάνουμε τι; Η απάντηση. Είναι ξεκάθαρη.Είναι για να μην κάνουμε τίποτα. Γιατί εμείς τι λέμε 2 χρόνια τώρα καλούμε όλα τα προοδευτικά κόμματα όσα αυτοχαρακτηρίζονται προοδευτικά. Ελάτε να μιλήσουμε, ελάτε να κάνουμε μια συζήτηση χωρίς να τα βρούμε αναγκαστικά, να συζητήσουμε. Εσεις τι λετε για τα κόκκινα δάνεια; Λέτε; Να μην υπάρχουν να μπει φρένο στους πλειστηριασμούς υποσχέθηκε, λέει ότι θα ακυρώσει το πλειστηριασμούς. Για πόσο; Τα ταμεία που έρχονται και αγοράζουν τα κόκκινα δάνεια. Έτσι έχουν εγγυήσεις 23 δισεκατομμύρια από εμάς εάν τους απαγορευτεί στους πλειστηριασμούς δεν θα βγάλουνε, οπότε θα πάμε, τι θα πάρουμε; Θα πάμε στην Λαγκάρντ να δανειστούμε 23 δισεκατομμύρια για να δώσουμε στα ταμεία επειδή θα έχουμε σταματήσει τους πλειστηριασμούς, τα πράγματα είναι απλά. Αν καθόμασταν γύρω από ένα τραπέζι, θα τους λέγαμε στους φίλους του ΣΥΡΙΖΑ, παιδιά, δεν γίνεται να σταματήσουμε τους πλειστηριασμούς.Εκτός αν θέλετε να δώσουμε 23 δις από χρήματα που δεν έχουμε στα ταμεία να τα πάνε στα Κέιμαν Άϊλαντς, οπότε. Είτε θα καταργήσουμε τον Ηρακλή. Μαζί με αυτή την εγγύηση των 23 δισεκατομμυρίων με νόμο θα το καταργήσουμε; Δεν σας χρωστάμε πια γιατί γιατί έτσι καμία Βουλή δεν μπορεί. Να δεσμεύσει μια επόμενη Βουλή; Βασική αρχή της δημοκρατίας. Και προστατεύουμε έτσι την κύρια κατοικία και το μικρομάγαζο ή δεν μπορούμε να τα προστατεύσουμε; Θα πρέπει να τους αφήσουμε να κάνουν τους πλειστηριασμούς για να βγάλουν τα 23 δισ. Ή να τους τα δώσουμε; Τα 23 δις, Λοιπόν, αυτή τη συζήτηση θέλαμε να κάνουμε εμείς και να δημιουργήσουμε ένα σφυρηλατήσουμε μια ενότητα στη βάση μιας κοινής λύσης.
