Και βέβαια, για να μην ξεχνάμε, αυτή είναι η κυβέρνηση, η οποία μπόρεσε και έβγαλε επιτέλους τις βαριοπούλες από τα πανεπιστήμια. Διότι εμείς είμαστε με τις βιβλιοθήκες και όχι με τις βαριοπούλες. Και όποτε χρειάστηκε η Αστυνομία να παρέμβει εδώ στου Ζωγράφου για να βγάλουμε από μέσα στοιχεία τα οποία δεν είχαν να κάνουν με τα πανεπιστήμια, δεν διστάσαμε να το κάνουμε.Θα είμαστε εμείς που θα εξασφαλίσουμε την επόμενη μέρα ότι τα πανεπιστήμιά μας θα είναι πραγματικές εστίες γνώσης, ότι τα παιδιά τα οποία δουλεύουν σκληρά σήμερα για να δώσουν πανελλαδικές εξετάσεις δεν θα εξαπατηθούν από κάποιους που τους λένε σήμερα «ελάτε, μωρέ, μπείτε αβέρτα στα πανεπιστήμια χωρίς να χρειαστεί καν να δώσετε εξετάσεις». Δεν τιμά αυτή η προσέγγιση όλα τα νέα παιδιά τα οποία δουλεύουν σκληρά σήμερα για να περάσουν στη σχολή την οποία θέλουν.Τα πανεπιστήμιά μας, τα δημόσια πανεπιστήμιά μας, θα αποκτήσουν τις υποδομές που χρειάζονται, θα ανοίξουν τα φτερά τους στον κόσμο, θα συνδεθούν επιτέλους με τον κόσμο της παραγωγής, θα προσφέρουν πτυχία με αντίκρισμα. Αυτό θέλουν τα νέα παιδιά κι αυτό θέλετε και εσείς οι οικογένειες, που έχουν επενδύσει στη δημόσια δωρεάν παιδεία, την οποία η δικιά μας κυβέρνηση απέδειξε ότι μπορεί να στηρίξει.
