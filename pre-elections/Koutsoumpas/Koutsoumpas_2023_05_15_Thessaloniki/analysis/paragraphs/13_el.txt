Αλλά πως να έχεις επαφή με τα προβλήματα του λαού όταν σε χωρίζει τέτοια απόσταση από όλα αυτά όσες μπαρούφες λοιπόν κι αν πουν δεν μπορεί να κρυφτεί ότι παντού ο λαός μας συναντά συνεχώς την ίδια αντίφαση, να υπάρχουν εκατομμύρια εργαζομένων που με τη δουλειά τους, τις γνώσεις τους, τις ικανότητές τους παράγουν όλον τον πλούτο αυτής εδώ της χώρας, να υπάρχουν εργαζόμενοι ικανοί να παράγουν, να δημιουργήσουν με υψηλή ειδίκευση και ικανότητα που δεν αξιοποιούνται όμως στον κλάδο τους, μια νέα γενιά που έρχεται με ορμή που παλεύει για τη μόρφωσή της, με όρεξη να δημιουργήσει και να μεγαλουργήσει, να αξιοποιήσει τις νέες τεχνολογίες και όσα έχει παράξει η ανθρωπότητα και από την άλλη μεριά να βρίσκεται μια χούφτα ανθρώπων μαζί με τις κυβερνήσεις τους, που όλη αυτή τη δύναμη πασχίζουν να την περιορίσουν, να τη φρενάρουν, να την υποτάξουν μέσα στα πλαίσια του δικού τους κέρδους που βλέπουν τους εργαζόμενους μόνο ως κάποια απαραίτητα εξαρτήματα και εργαλεία για να πετύχουν τα περί πολυπόθητα αυτά καπιταλιστικά τους κέρδη, που βλέπουν τα δικαιώματα του λαού μας ως εμπόδιο για τα κέρδη τους. Από τη μια είναι μια χούφτα ανθρώπων και οι εκπρόσωποί τους που μιλούν μια ξύλινη, μια ακαταλαβίστικη γλώσσα για δημοσιονομικούς κανόνες, για ανταγωνιστικότητα, για επιλέξιμες ημιδαπάνες για δημοσιονομικά περιθώρια.
