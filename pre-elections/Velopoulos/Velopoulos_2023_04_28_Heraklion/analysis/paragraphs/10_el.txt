Δυστυχώς αυτό δείχνει ότι περνάμε σε επικίνδυνα μονοπάτια καθεστωτικών επιλογών και είπα και προηγουμένως δεν μπορείς να απαγορεύεις κόμματα επειδή χρησιμοποιήσαν ένα σύμβολο το οποίο είναι ελληνικό, δεν είναι άλλης χώρας ή μία λέξη ή δύο λέξεις και να τους απαγορευείς να κατέβουν σε εκλογές. Είναι ντροπή αυτό που γίνεται. Όπως ντροπή είναι ένα κόμμα να καταγγέλλει άλλα κόμματα και να φτάνουν στο σημείο να σκοτώνονται μεταξύ τους οι αρχηγοί ή τα στελέχη των κομμάτων για μια θέση. Σταματώ εδώ, γιατί τα πράγματα είναι πολύ πιο δύσκολα για τις εκλογές, διότι ακούστε. Ένα εκατομμύριο έλληνες ομογενείς είχαν τη δυνατότητα περίπου, γιατί υπάρχουν, να ψηφίσουν, εάν η πολιτεία τους βοηθούσε. Από αυτούς μόνο οι εικοσιπέντε χιλιάδες θα ψηφίσουν, ομογενείς, και θα πάνε να ψηφίσουν γιατί γράφτηκαν σε μερικούς καταλόγους. Το περίεργο είναι ότι οι Έλληνες της Αργεντινής ψηφίζουν στο Χιούστον των ΗΠΑ. Δηλαδή θα τους αναγκάσουν να φύγουν από την Αργεντινή και να πάνε να ψηφίσουν χιλιάδες χιλιόμετρα μακριά, να πάνε στο Χιούστον. Οι βορειοηπειρώτες μπορούν να ψηφίσουν στη Σόφια. Οι Έλληνες του Περθ ψηφίζουν στη Μελβούρνη τρεισήμισι χιλιάδες χιλιόμετρα μακριά. Αυτό δεν είναι ψήφος των ομογενών, οι οποίοι πρέπει να ψηφίζουν, πρέπει να συμμετέχουν, πρέπει να παρεμβαίνουν, γιατί η ομογένεια είναι πολύ σημαντική για την Ελλάδα. Για να έχει εξωστρέφεια, έκανε τα αδύνατα δυνατά η Νέα Δημοκρατία για να δημιουργήσει προβλήματα στους ομογενείς μας. Τα λεω αυτά γιατί και οι Κρητικοί έχουν ομογενείς σε όλο τον κόσμο, κυρίως στην Αμερική, εξ όσων γνωρίζω και δυστυχώς αυτή η πολιτεία δεν βοηθάει τους Έλληνες ομογενείς να ψηφίσουν. Θα το πω διαφορετικά, η κυβέρνηση κάνει τα αδύνατα δυνατά ώστε να μην ψηφίσουν. Ίσως γιατί φοβάται την αγάπη για τον νόστο που έχουν ομογενείς για την Ελλάδα, για τα εθνικά θέματα, την ιστορία και τον πολιτισμό και θέλει να τους αποτρέψει να παν στις κάλπες.
